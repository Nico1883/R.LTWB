Create audio files from text - TTS using Microsoft Edge
	1. Create a Microsoft Word document with the TTS text script.
	2. Convert the .docx file to an .pdf file.
	3. Open the .pdf file with Microsoft Edge.
	4. Open the Window Sound Recorder and choose the Stereo Mix sound system. Start the record.
	5. En Microsoft Edge, clic the option Read Aloud and let read all the file. Voice: Microsoft Gonzalo
	5. Stop the recording.
